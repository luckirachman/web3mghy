CARA CEK KELULUSAN via WhatsApp

Oleh : luckirachman diterbitkan tanggal 05 Juni 2020

Pengumuman Kelulusan Tahun 2019/2020 di SMPN 3 Margahayu, apabila kebijakan Lockdown/PSBB masih diberlakukan maka akan dilaksnakan secara online melalui aplikasi WhatsApp (Download Apps). Surat Kelulusan diberikan dalam bentuk lembar SK Kelulusan berupa file .pdf, siswa dapat langsung mengunduh file tersebut dan dapat menyimpannya ke dalam perangkat (HP/Laptop/Komputer).

Untuk mendapatkan informasi kelulusan tersebut, dapat dilakukan langkah-langkah berikut:

1. Ketikan NISN Kamu (Nomor Induk Siswa) misal, 123456789 *tidak tahu, tanyakan ke Wali Kelas 

2. Lalu kirim ke No. WA 0821-1385-1488 (SMPN 3 Margahayu)

3. Tunggu balasan muncul secara otomatis

4. Muncul balasan berupa link Google Drive berisi file .pdf Surat Kelulusan

5. Klik link Surat Keterangan Lulus tersebut, pilih aplikasi untuk membuka, misal PDF Viewer

6. Lalu akan muncul file Surat Keterangan Lulus dalam bentuk .pdf

7. Terakhir untuk mengunduh (download), klik menu / ikon unduh di pojok kanan atas.

Catatan :

*Pengumuman SKL :

     dibuka mulai tanggal 5 Juni 2020 pukul 06.00

     ditutup pada tanggal 6 Juni 2020 pukul 24.00.
